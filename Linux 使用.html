博客园看来的

注: 下面的 向后 指光标左边; 向前 指光标右边

  • 移动
    1. Ctrl+a : 移动到当前行的开头
    2. Ctrl+e : 移动到当前行的结尾
    3. Ctrl+b : 向后移动一个字符
    4. Ctrl+f : 向前移动一个字符
    5. Alt +b : 向后移动一个单词
    6. Alt +f : 向前移动一个单词
  • 清屏
    • Ctrl+l
  • 编辑
    1. Ctrl+u : 删除光标所在处之前的所有字符(不包括自身)
    2. Ctrl+k : 删除光标所在处之后的所有字符(包括自身)
    3. Ctrl+d : 删除光标所在处字符
    4. Ctrl+h : 删除光标所在处前一个字符
    5. Ctrl+w : 删除光标所在处之前的字符至其单词头(以空格,标点等为分隔符)
  • 粘贴
    • Ctrl+y : 粘贴刚才所删除的字符
  • 撤销
    • Ctrl+(x u) : 按住Ctrl的同时再先后按x和u, 撤销刚才的操作
