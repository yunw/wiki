编译时加入stabs格式的调试信息,一般以汇编伪指令方式出现.

stabs调试信息的汇编代码形式

  • stabs "strings", type, other, desc, value
    反映源程序的文件名, 函数, 变量等相关属性以及所使用的编程语言的类型信息
  • stabn type, other, desc, value
    反映源程序的行号信息,程序块结构信息等
  • stabd type, other, desc
  • stabx "strings", value, type, sdb-type

type 取值(0~0xff), 其决定了该条伪指令所表示的信息是否为stabs信息, 也决定了其余 字段的内容. "strings" - "name: symbol-descriptor type-information" name: 表示该符号名字的字符串(为空串时表示没有名字的对象) symbol-descriptor: 一英文字母, 表示该符号是哪一种符号信息(f-文件内函数, F-全局 函数, G-全局变量). type-information: 表示类型的二元整数值对(xxx, xxx), 或"(xxx,xxx)=xxx"的形式. (xxx, xxx)表示直接指向该二元整数值对所代表的已定义的类型, 否则为新类型
