提交本地分支到远程仓库中

将分支jj07提交到名为aosp的远程仓库里的jj07分支 git push aosp jj07:jj07

删除远程仓库中的分支

删除远程仓库aosp中的jj07分支 git push aosp :jj07

设置本地分支跟踪远程分支

git config branch.jj07.remote aosp git config branch.jj07.merge refs/heads/jj07

这样设置后，在本地的jj07分支中，使用 git pull/push 会自动与远程仓库aosp中的jj07分支进行数据同步
