我的debian设置 

if [ "$color_prompt" = yes ]; then
    PS1='${debian_chroot:+($debian_chroot)}\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[00m\]\$ '
else
    PS1='${debian_chroot:+($debian_chroot)}\u@\h:\W\$ '
fi

Bash使用PS1~PS4来显示提示符号，其格式如下 \t : 现在时间。 \d : 现在日期。 \n : 新行。 \s : shell的名称。 \w : 目前工作目录完整路径。 \W : 目前工作目录。 \u : 使用者名称。 \h : Hostname。 \# : 这个命令的号码。 \! : 历史号码。 \$ : 如果EUID是0，则#，否则为$。 \\ : "\"符号。
